# Smith Button - Quick Start Guide

## Step 1: Load the Extension

1. Open Google Chrome
2. In the address bar, type: `chrome://extensions/` and press Enter
3. In the top right corner, toggle **Developer mode** ON (it will turn blue)
4. Click the **Load unpacked** button (top left)
5. Navigate to and select this folder: `c:\Users\mikem\Code\Smith_Button`
6. Click **Select Folder**

The Smith Button should now appear in your extensions list and toolbar!

## Step 2: Pin the Extension (Optional but Recommended)

1. Click the puzzle piece icon (🧩) in your Chrome toolbar
2. Find "Smith Button" in the list
3. Click the pin icon next to it
4. The Smith Button icon will now be visible in your toolbar

## Step 3: Test It Out

1. Go to a recipe website (try: https://www.allrecipes.com/recipe/12151/banana-banana-bread/)
2. Click the Smith Button icon in your toolbar
3. The side panel will open with the extracted recipe!

## Step 4: Set Up API Key (Optional)

Many recipe sites work without an API key. But for full functionality:

1. Click the Smith Button
2. If prompted, click "Set Up API Key"
3. Get a key from: https://platform.openai.com/api-keys
4. Paste it and click "Save API Key"

## Troubleshooting

**Don't see the Smith Button icon?**
- Check if the extension loaded successfully in `chrome://extensions/`
- Look for any error messages in red
- Try clicking the puzzle piece icon (🧩) to see all extensions
- Pin the extension to make it always visible

**Extension won't load?**
- Make sure Developer mode is ON
- Verify you selected the correct folder (`Smith_Button`)
- Check that all files are present (especially the `icons` folder)

**Still having issues?**
- Look at the extension card in `chrome://extensions/`
- Click "Errors" if there's a red button
- Check the browser console (F12) for error messages
